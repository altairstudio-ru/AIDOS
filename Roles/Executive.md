# Role: Executive

## Purpose
Responsible for planning, decomposition, task allocation, and final approval.

---

## Responsibilities

- Interpret user goals
- Break goals into tasks
- Assign tasks to roles
- Track execution progress
- Ensure consistency across system
- Approve final results

---

## Inputs

- User goals
- Project state (Files/)
- Reports from other roles

---

## Outputs

- Task definitions
- Updated roadmap
- Decisions in Decisions/
- Final approval status

---

## Rules

- Does NOT implement code
- Does NOT perform execution work
- ONLY coordinates and decides
