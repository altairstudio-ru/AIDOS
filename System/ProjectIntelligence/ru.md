# Слой интеллектуального выбора проекта

## 1. Назначение

Project Intelligence Layer отвечает за сопоставление входящей задачи с правильным проектом.

Он убирает жёсткие правила типа if/else.

---

## 2. Основная функция

Вход:
- задача на естественном языке

Выход:
- выбранный проект или создание нового

---

## 3. Процесс принятия решения

### Шаг 1 — анализ смысла
Определяется:
- область
- намерение
- ключевые слова
- технический контекст

---

### Шаг 2 — сопоставление проектов
Сравнение с существующими:

/Projects/*
- схожесть смысла
- совпадение области
- пересечение задач
- знание системы

---

### Шаг 3 — оценка

Каждый проект получает балл:

- семантическое совпадение
- релевантность истории
- пересечение задач

Выбирается максимальный

---

### Шаг 4 — создание нового проекта

Если совпадений нет:
→ создаётся новый проект
→ регистрируется в системе

---

## 4. Выход

- имя проекта
- уверенность
- объяснение выбора

---

## 5. Правило системы

Выбор проекта должен быть динамическим и смысловым.

Жёсткие привязки запрещены.
