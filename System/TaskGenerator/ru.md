# Генератор задач

## 1. Назначение

Task Generator преобразует высокоуровневые цели в первичный набор структурированных задач.

Это точка входа автоматизации системы.

---

## 2. Входные данные

- цель пользователя
- контекст проекта
- текущее состояние системы

---

## 3. Выход

- начальный набор задач TASK-001, TASK-002...
- готовые файлы задач
- назначенные роли для каждой задачи

---

## 4. Логика генерации

### Шаг 1 — анализ цели
Определяется:
- что нужно сделать
- ограничения
- область задачи

---

### Шаг 2 — определение типов работ
Классификация:
- исследование
- разработка
- проверка
- проектирование

---

### Шаг 3 — построение графа задач
Определяются:
- зависимости
- порядок выполнения
- возможность параллельной работы

---

### Шаг 4 — создание задач
Формируются задачи:

- TASK-001
- TASK-002
- TASK-003

каждая как отдельный файл

---

## 5. Отличие от Router

- Generator создаёт начальную структуру задач
- Router управляет их распределением в процессе работы

---

## 6. Итоговое правило

Task Generator превращает цель в стартовое состояние системы задач.
