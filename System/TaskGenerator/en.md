# Task Generator

## 1. Purpose

Transforms high-level goals into initial structured task sets.

It is the entry point of system automation.

---

## 2. Input

- User goal
- Project context
- Existing system state

---

## 3. Output

- Initial Task-001 set
- Execution-ready task files
- Assigned roles per task

---

## 4. Generation Logic

### Step 1 — Understand Goal
Parse:
- objective
- constraints
- domain

---

### Step 2 — Identify Work Types
Classify:
- research
- implementation
- validation
- design

---

### Step 3 — Create Task Graph
Define:
- dependencies
- sequence
- parallelism

---

### Step 4 — Emit Tasks
Generate:
- TASK-001
- TASK-002
- TASK-003

as physical files

---

## 5. Difference from Router

- Generator creates initial structure
- Router manages ongoing flow

---

## 6. Final Principle

Task Generator turns intent into initial system state.
